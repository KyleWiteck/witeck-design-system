A collapsible panel that helps organize content into sections. It is
**accessible, animated, and customizable**, making it ideal for **FAQs, settings
panels, and content toggles**.

It supports:

- Dynamic expansion/collapse with animation
- Keyboard accessibility
- Customizable sizes and title elements

## How to Use

### Basic Example

By default, the **content is hidden** and expands when clicked, To start with an
**expanded section**, use `defaultOpen`.

Accordions function independently, allowing users to **open multiple sections**.

```tsx live
<Accordion title="Accordion Base">
  <Stack gap="6">
    <Text>
      Lorem ipsum dolor sit amet, consectetur adipisicing elit. Tenetur itaque
```

## Keyboard & Accessibility Features

- **Enter / Space**: Toggles the accordion.
- **Tab Key**: Navigate between multiple accordions.
- **ARIA Attributes** ensure screen reader support.

## Best Practices

- Use meaningful `title` labels.
- Maintain logical heading structure with `titleElement`.
- Avoid nesting accordions within each other.
- Keep content concise inside the accordion.

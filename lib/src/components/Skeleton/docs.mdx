Loading indicator component with customizable size and color variants. Built for
consistent loading states across the application.

## Usage

```tsx
import { Skeleton } from '@peopleticker/magnit-design/components';

// Basic usage
<Skeleton height="20" width="full" />;

// Image variant
<Skeleton variant="image" boxSize="32" />;
```

## Card Loading

```tsx live
function CardLoading() {
  return (
    <Card padding="4" width="full" minHeight="60">
      <Stack gap="4">
        <Flex gap="4">
          <Skeleton variant="image" boxSize="16" />
          <Stack gap="2" flex="1">
            <Skeleton height="6" width="3/4" />
            <Skeleton height="4" width="1/2" />
          </Stack>
        </Flex>
        <Skeleton height="4" width="full" />
        <Skeleton height="4" width="full" />
        <Skeleton height="4" width="2/3" />
      </Stack>
    </Card>
  );
}
```

## Best Practices

- Match skeleton dimensions as close to actual content, this improves UX.
- Use consistent spacing between skeleton elements
- Apply appropriate shade based on background color (dark is secondary)
- Maintain content hierarchy in skeleton layout
- Consider mobile viewport loading states

## Technical Details

- Built on `Center` component
- Customizable animation timing
- ARIA busy state management
- SVG-based image placeholders
- Responsive dimension support
- RTL layout compatible
- Minimal re-renders with memo

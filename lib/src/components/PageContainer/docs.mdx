A layout component that provides consistent page structure with optional footer
support and responsive container sizing.

## Usage

```tsx
import { PageContainer } from '@KyleWiteck/witeck-design/components';

<PageContainer footer={<Footer />}>
  <Content />
</PageContainer>;
```

```tsx live

</PageContainer>
```

## With Footer

```tsx live
function PageWithFooter() {
  return (
    <PageContainer border="1px" footer={<Footer />}>
      <Stack gap="4">
        <Text>Main content</Text>
        <Text>Scrollable area</Text>
      </Stack>
    </PageContainer>
  );
}
```

### Custom Container Size

```tsx live
<PageContainer size="lg" border="1px">
  <Card>
    <Text>Constrained width content</Text>
  </Card>
</PageContainer>
```

## Best Practices

- Set a specific size prop for content width constraints instead of handling
  widths in children
- Use `wrapperStackProps` for layout adjustments rather than wrapping in
  additional containers
- Maintain consistent vertical padding across pages using the default `paddingY`
  value
- Place sticky elements (navigation, toolbars) outside `PageContainer` to
  prevent scroll conflicts
- Consider mobile viewport heights when implementing footers

## Technical Considerations

- Full viewport height layout
- Automatic content scrolling
- Footer pinned to bottom
- Responsive padding
- Polymorphic component support
- Memoized for performance

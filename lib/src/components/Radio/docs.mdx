## Basic Usage

```tsx live
function BasicRadio() {
  return (
    <Stack gap="2">
      <Radio id="option1" name="group" label="Option 1" />
      <Radio id="option2" name="group" label="Option 2" />
    </Stack>
  );
}
```

## Form Integration

```tsx live
function RadioForm() {
  const [value, setValue] = useState('');

  return (
    <form>
      <Stack gap="4">
        <Radio
          id="radio1"
          name="options"
          value="option1"
          checked={value === 'option1'}
          onChange={e => setValue(e.target.value)}
          label="First Option"
        />
        <Radio
          id="radio2"
          name="options"
          value="option2"
          checked={value === 'option2'}
          onChange={e => setValue(e.target.value)}
          label="Second Option"
          hasError={!value}
        />
      </Stack>
    </form>
  );
}
```

## Best Practices

- Use unique IDs for each radio button
- Group related options with same `name` prop
- Provide clear, concise labels
- Handle form validation appropriately
- Consider keyboard navigation
- Use disabled state sparingly

## Technical Details

- Built on native radio input
- ARIA-compliant
- Supports form validation
- Customizable styling
- RTL support
- Keyboard accessible

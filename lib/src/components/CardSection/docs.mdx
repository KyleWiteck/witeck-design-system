Customizable card for presenting structured content section. This CardSection
component provides flexibility for creating structured, accessible card layouts
with optional loading states, ideal for dashboards, profile cards, or similar UI
use cases.

## Usage

Import the `CardSection` component and use it in your application with the
desired props.

```tsx
import { CardSection } from '@KyleWiteck/witeck-design/components';

<CardSection
  title="Heading Title"
  subtitle="Section's subtitle"
  actions={<Button size="sm">Primary Action</Button>}
  variant="column"
>
  <Text>This is the content of the card.</Text>
</CardSection>;
```

## Technical Considerations

- Padding is added automatically when variant is set to 'column'.
- Adjusts the layout dynamically based on the variant or isLoading state.
- Uses Skeleton components to display placeholders when isLoading is true.

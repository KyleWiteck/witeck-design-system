The Text component is a fundamental typography element that provides consistent
text styling with support for various styles, weights, and alignments.

## Usage

```tsx
import { Text } from '@KyleWiteck/witeck-design/components';

<Text variant="h1" element="h1">Heading</Text>
<Text variant="body1" color="gray-600">Body text</Text>
```

## Custom Styling

```tsx live
function CustomStyling() {
  return (
    <Text
      variant="custom"
      color="primary"
      align="center"
      textDecoration="underline"
      fontStyle="italic"
    >
      Custom styled text
    </Text>
  );
}
```

## Best Practices

- Use semantic variants for proper document hierarchy
- Use the correct element for proper semantic markup.
- Maintain consistent styling within related content
- Apply appropriate text alignment for content context
- Consider color contrast for accessibility.
  [Color Contrast Checker](https://coolors.co/contrast-checker/112a46-acc8e5)

## Technical Details

- Built on Box component
- Polymorphic component support
- RTL compatible
- Accessible by default

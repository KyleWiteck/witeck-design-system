A radio button styled as a chip component. It supports keyboard navigation,
accessibility, and can be used inside a ChipRadioGroup.

## Features

- Styled as a chip component for a modern UI.
- Fully accessible with aria-checked`, keyboard support, and roving tabindex.
- Should be used inside `ChipRadioGroup` for managing selection.
- **Only supports controlled** usage, via `checked` and `onChange` props.

### ChipRadioGroup

The `ChipRadioGroup` is a container component that manages a group of
`ChipRadioButton` components. It ensures only one chip can be selected at a
time, following the WAI-ARIA Radio Group pattern.

## Usage

```tsx live
function Component() {
  const [value, setValue] = useState('1');
  return (
    <ChipRadioGroup
      name="chip-radio-test"
      onChange={e => setValue(e.target.value)}
    >
      <ChipRadioButton value="1" checked={value === '1'}>
        First
      </ChipRadioButton>
      <ChipRadioButton value="2" checked={value === '2'}>
        Second
      </ChipRadioButton>
      <ChipRadioButton value="3" checked={value === '3'}>
        Third
      </ChipRadioButton>
    </ChipRadioGroup>
  );
}
```

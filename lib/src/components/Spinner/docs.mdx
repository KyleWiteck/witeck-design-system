Loading indicator component with customizable size and color variants. Built for
consistent loading states across the application. Best used when `Skeleton`
component is not a good use case.

## Usage

```tsx
import { Spinner } from '@KyleWiteck/witeck-design/components';

<Spinner size="md" color="primary" />;
```

## Size Variants


## Best Practices

- Use consistent sizes within similar contexts
- Match spinner color to surrounding elements
- Provide loading text for accessibility
- Maintain balanced spacing

## Technical Details

- CSS animation-based rotation
- Center component alignment
- Accessible loading state
- Minimal performance impact

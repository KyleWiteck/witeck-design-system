## Basic Usage

```tsx live
<Flex gap="4">
  <Box padding="4" backgroundColor="primary100">
    Item 1
  </Box>
  <Box padding="4" backgroundColor="primary100">
    Item 2
  </Box>
  <Box padding="4" backgroundColor="primary100">
    Item 3
  </Box>
</Flex>
```

  </Box>
  <Box flex="1" padding="4" backgroundColor="primary100">

## Technical Considerations

- Inherits all Box props
- Supports responsive values
- Maintains proper spacing with gap utility
- Handles nested flex containers
- Preserves proper HTML semantics
- Polymorphic element support
- Defaults to `flexDirection: 'row'`

## Best Practices

- Use semantic HTML elements when appropriate
- Consider mobile-first responsive layouts
- Maintain consistent spacing patterns
- Use gap instead of margins for spacing
- Consider flex-wrap for responsive content

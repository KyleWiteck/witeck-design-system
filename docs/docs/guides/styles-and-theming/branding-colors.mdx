## Customization Methods

### Static (CSS)

```css
/* styles/theme.css */
:root {
  --primary__color: #ff0000; /* Override default #293F96 */
}
```

### CSS Module

```css
/* styles/theme.module.css */
.theme {
  --primary__color: #ff0000;
}
```

### Runtime (JavaScript)

```typescript
// Override primary color
document.documentElement.style.setProperty('--primary__color', '#FF0000');

// Reset to default
document.documentElement.style.removeProperty('--primary__color');
```

### React Component

```tsx
function BrandThemeProvider({ primaryColor, children }) {
  useEffect(() => {
    if (primaryColor) {
      document.documentElement.style.setProperty(
        '--primary__color',
        primaryColor
      );
    }
    return () =>
      document.documentElement.style.removeProperty('--primary__color');
  }, [primaryColor]);

  return children;
}
```

:::info Default Values

- Primary color defaults to `#293F96` when not overridden
- Secondary color customization is planned for future releases :::

:::tip Best Practices

- Use CSS files for static brand colors
- Use JavaScript for dynamic theme changes
- Set colors at root level
- Test color contrast for accessibility :::
